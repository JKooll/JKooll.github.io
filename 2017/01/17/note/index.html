Linux挂在NTFS格式硬盘

Jan 17, 2017 | 笔记Linux | Hits

问题(Question)

linux一般不支持NTFS文件系统，在挂在硬盘时，会出现只读现象，可以通过安装ntfs-3g来实现在linux上读写NTFS格式的硬盘。

准备(Prepare)

  • 安装有linux的电脑或虚拟机
  • apt-get
  • vim
  • 准备要挂在的NTFS硬盘

开始(Begin)

安装ntfs-3g

sudo apt-get install ntfs-3g

查看要挂在的NTFS文件系统硬盘

sudo fdisk -l | grep NTFS

挂载

手动挂载

mount -t ntfs-3g /dev/ /{mnt,…}/

例如：mount -t ntfs-3g /dev/sdb1 /mnt/d

或

ntfs-3g /dev/ /mnt/

例如：ntfs-3g /dev/sda1 /mnt/c

启动时挂载

首先使用vim打开/etc/fstab,sudo vim /etc/fstab,按照 ntfs-3g defaults 0 0，将要挂在的硬盘信息插入到文件中。例如：/dev/sdb1 /mnt/e ntfs-3g defaults 0 0

Q & A

配置完成之后发现挂在的ntfs硬盘还是只读，原因可能是挂在的硬盘NTFS文件系统损坏，修复可以使用windows的磁盘检查工具，还可以通过ntfsfix修复，例如想要修复/dev/sdb2，先卸载硬盘umount /dev/sdb2, 修复ntfsfix /dev/sdb2, 重新挂载硬盘mount /dev/sdb2

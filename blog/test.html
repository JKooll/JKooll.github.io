# 使用cookie实现跨页面认证 （PHP版本） ## 介绍 本文通过cookie实现用户登录 ## 创建一个文件 -- index.php， 将下面的代码复制到index.php中 
        
            <!DOCTYPE html&rgt;
            <html&rgt;
        
            <head&rgt;
                <title>Auth Cookie</title&rgt;
            </head&rgt;
        
            <body&rgt;
                <form method="get" action="index.php"&rgt;
                    <input type="text" name="action" value="login" hidden&rgt;
                    <input type="text" name="username"&rgt;
                    <input type="text" name="password"&rgt;
                    <input type="submit" value="Submit"&rgt;
                </form&rgt;
                <button&rgt;<a href="main.php?action=logout">Logout</a&rgt;</button&rgt;
            </body&rgt;
        
            </html>
